## ğŸ†˜ Troubleshooting

**Q: My data disappeared!**
A: Check if you cleared your browser data. Always keep backups using the "Backup Data" feature.

**Q: Can I use this on multiple devices?**
A: Yes! Use "Backup Data" on one device and "Import Data" on another.

**Q: How do I create a PDF?**
A: Click "Export PDF" to download an HTML file. Open it in your browser and use Print > Save as PDF.

**Q: Is there a limit to how much I can write?**
A: The limit depends on your browser's localStorage capacity (typically 5-10MB), which is enough for thousands of entries.

**Q: Can I share my memoir with others?**
A: Yes! Export to PDF and share the file, or use the backup JSON file.
